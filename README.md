# ⚡ Chain Signal AI

> Bloomberg Terminal meets Web3 — a real-time AI signal engine for on-chain crypto intelligence.

---

## 🧠 Overview

**Chain Signal AI** is a real-time, AI-powered on-chain signal analysis dashboard that scans live blockchain data and classifies market conditions using large language model intelligence. Built with a professional trading terminal aesthetic — dark cyberpunk UI, CRT scanlines, glowing signal badges — it delivers actionable market signals that a serious trader would actually trust.

Feed it a token name or contract address. It fetches live block data from the **OneChain Mainnet RPC**, feeds that context into **Groq's LLaMA 3 70B** model, and returns a structured signal card with momentum classification, whale activity detection, risk level, and a 3-sentence AI market narrative — all in under 3 seconds.

---

## ✨ Features

- 🤖 **AI-generated market analysis** with animated `[ AI ANALYSIS ]` prefix and blinking cursor
- 📊 **Risk classification system:**
  - 🟢 **Bullish / Low Risk** — green glow
  - 🔴 **Bearish / High Risk** — red glow
  - 🟡 **Neutral / Medium Risk** — amber glow
  - 🐋 **Whale Activity Detection** — cyan glow (large transaction tracking)
- 🔗 **Real-time data scanning** from OneChain Mainnet RPC (`https://rpc-mainnet.onelabs.cc:443`)
- ⏳ **Terminal-style scan loader** with animated scanlines and staggered data-fetch messages
- 📜 **Recent Signals history** — last 3 analyses kept in session with color-coded momentum bars
- 🖥️ **Cyberpunk trading terminal UI** — CRT scanline overlay, pulsing orbs, dot grid, glassmorphism panels
- 🔴 **LIVE indicator** — fixed top-right status chip showing live OneChain connection

## 🔧 Usage

### Development

```bash
# Start the API server
pnpm --filter @workspace/api-server run dev

# Start the frontend (in a separate terminal)
pnpm --filter @workspace/onesignal run dev
```

The dashboard will be available at `http://localhost:25107` (or the port printed by Vite).

The API server runs at `http://localhost:8080/api`.

### What you'll see

1. **Enter a token name** (e.g. `ETH`, `ONE`, `USDC`) or paste a contract address in the search bar
2. Click **ANALYZE >** — the terminal loader fires up and scans OneChain in real time
3. A **Signal Card** appears with:
   - Momentum badge (Bullish / Neutral / Bearish) with color glow
   - Whale Activity level (High / Medium / Low)
   - Risk Level with corresponding glow intensity
   - A 3-sentence AI narrative generated by LLaMA 3 70B
4. Your last **3 analyses** are saved as Recent Signals below the card

### Regenerating API types

If you modify the OpenAPI spec at `lib/api-spec/openapi.yaml`:

```bash
pnpm --filter @workspace/api-spec run codegen
```
